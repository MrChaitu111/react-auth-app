# React Auth App

This project is a simple, professional authentication app built with **React**, **React Router DOM**, and **Tailwind CSS**. It includes:

- Signup and Login pages
- Password validation (8+ chars, uppercase, lowercase & number)
- Show/Hide password functionality with toggle icons
- LocalStorage-based authentication
- Clean, minimal black-and-white design
- Fully responsive UI

### Tech Stack
- React
- React Router DOM
- Tailwind CSS
- react-toastify (for notifications)
